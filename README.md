# Laboratorio-_Ecosistema_Odoo

El trabajo nos pedía que realizásemos diferentes gestiones dentro de la aplicación de Odoo. El primer paso fue instalar las aplicaciones de Ventas y Facturación, que en mi caso ya estaban instaladas. Más adelante, creé un archivo llamado clientes_mock.csv donde pegué el código que se nos dio en la actividad e instalé la extensión de Rainbow CSV. 

Una vez hecho esto, dentro de Odoo, importé este archivo dentro de la sección Ventas>Pedidos>Clientes, validándolo más adelante. Tras esto, accedí desde la cuenta de comercial que creamos el jueves y creé un nuevo presupuesto, donde encontré un problema a la hora de añadir un producto a este presupuesto
